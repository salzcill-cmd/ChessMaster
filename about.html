🎯 Tentang ChessMaster Arena

Platform bermain catur premium yang dirancang untuk memberikan pengalaman terbaik bagi pecinta catur, dari pemula hingga master. Bermainlah melawan AI cerdas atau tantang teman Anda dalam mode hot seat.

Deskripsi Project

ChessMaster Arena adalah website game catur yang dibangun dengan teknologi web modern. Project ini bertujuan menyediakan platform bermain catur yang mudah diakses, ringan, dan menyenangkan tanpa perlu instalasi aplikasi.

Kami percaya bahwa catur adalah olahraga pikiran yang dapat melatih strategi, kesabaran, dan kemampuan berpikir kritis. Dengan ChessMaster Arena, siapa saja dapat berlatih dan bermain kapan saja.

Teknologi yang Digunakan

Project ini dibangun dengan stack teknologi berikut:

HTML5 CSS3 JavaScript (Vanilla) Bootstrap 5 Google Fonts Font Awesome

Fitur Teknis:

  • ✅ Multi-page HTML dengan navigasi internal
  • ✅ CSS & JavaScript internal (tanpa file eksternal)
  • ✅ Responsive design untuk semua device
  • ✅ Glassmorphism UI dengan animasi halus
  • ✅ Chess engine dengan validasi gerakan lengkap
  • ✅ AI dengan 3 tingkat kesulitan

Fitur Unggulan

  • ♟️ Papan Catur Interaktif: Drag & click, highlight gerakan valid, animasi halus
  • 🤖 AI Cerdas: 3 level kesulitan dengan strategi berbeda
  • 👥 Hot Seat Mode: Bermain 2 pemain di satu perangkat dengan flip board
  • ⏱️ Timer & Statistik: Pantau durasi permainan dan jumlah gerakan
  • 🎨 UI Premium: Dark theme dengan aksen emas, glassmorphism, dan efek visual menarik
  • 📱 Fully Responsive: Tampilan optimal di desktop, tablet, dan mobile
